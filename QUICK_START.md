# DexRadar Quick Start

## Get Running

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Routes

- `/` - Dashboard with token pairs
- `/pairs` - All pairs with search
- `/pair/[id]` - Pair detail page

## Features

- Multi-chain (Solana, Ethereum, BSC, Base, etc.)
- Real-time pair data (mock - integrate API)
- Wallet connect (Solana)
- Search and filter
